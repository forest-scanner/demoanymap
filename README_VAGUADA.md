# Análisis de Series Temporales del Parque La Vaguada

Este proyecto realiza un análisis de series temporales de datos simulados del Parque La Vaguada de Madrid.  Se generan datos de visitantes, temperatura y actividades, y se crean visualizaciones y un mapa interactivo.

## Características

*   **Generación de datos simulados:** Datos realistas de visitantes, temperatura y actividades.
*   **Análisis de series temporales:** Estadísticas, análisis por día de la semana y mes, correlación.
*   **Visualizaciones:** Gráficos con matplotlib.
*   **Mapa interactivo:** Mapa con folium.
*   **Reporte:** Reporte en formato texto.

## Archivos

*   `anymap_ts_parque_vaguada.py`: Código principal del análisis.
*   `mapa_parque_vaguada.html`: Mapa interactivo generado.
*   `visualizaciones_parque_vaguada.png`: Gráficos generados.
*   `reporte_parque_vaguada.txt`: Reporte del análisis.
*   `README.md`: Este archivo.

## Dependencias

*   pandas
*   numpy
*   matplotlib
*   folium

## Cómo ejecutar

1.  **Clonar el repositorio:** (asumiendo que se subirá a GitHub)
    ```bash
    git clone <URL del repositorio>
    cd <nombre del repositorio>
    ```

2.  **Instalar dependencias:**
    ```bash
    pip install pandas numpy matplotlib folium
    ```

3.  **Ejecutar el script:**
    ```bash
    python anymap_ts_parque_vaguada.py
    ```
