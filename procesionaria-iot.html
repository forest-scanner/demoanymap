Estudio de Procesionaria del Pino con IoT y LoRaWAN

Sistema de monitorización inteligente para la detección temprana y control de la plaga de procesionaria en zonas verdes urbanas

📊 Problema a Resolver

La procesionaria del pino (Thaumetopoea pityocampa) es una plaga forestal que afecta gravemente a:

  • Salud pública (alergias, urticarias)
  • Ecosistemas urbanos (deforestación, pérdida de biodiversidad)
  • Economía local (costes de control, turismo)
  • Seguridad en parques y zonas recreativas

🚀 Solución Propuesta

Sistema de monitorización distribuida basado en sensores IoT conectados mediante red LoRaWAN para:

🔍 Detección Temprana

Sensores especializados detectan actividad de orugas antes de que sean visibles a simple vista.

📡 Comunicación de Largo Alcance

Red LoRaWAN permite cobertura en grandes áreas con mínimo consumo energético.

📊 Análisis Predictivo

Algoritmos de IA analizan datos para predecir brotes y optimizar tratamientos.

🏗️ Arquitectura del Sistema

1. Sensores IoT

Cámaras térmicas, sensores de vibración y ambientales instalados en pinos

2. Gateway LoRaWAN

Recolecta datos de múltiples sensores y los envía a la nube

3. Plataforma Cloud

Procesamiento, almacenamiento y análisis de datos

4. Dashboard Web

Visualización en tiempo real y alertas para gestores municipales

🔧 Tecnologías Utilizadas

LoRaWAN ESP32 / Arduino Sensores IoT Python / TensorFlow Node-RED InfluxDB Grafana MQTT Docker GitHub Actions

✅ Beneficios del Sistema

💰
Reducción de Costes

Hasta 40% en tratamientos mediante intervención precisa

🌳
Protección Ambiental

Minimización de pesticidas, solo donde sea necesario

👥
Seguridad Ciudadana

Alertas tempranas para evitar zonas afectadas

📈
Datos para Investigación

Base de datos para estudios ecológicos a largo plazo

📅 Fases de Implementación

  1. Fase 1 (3 meses): Prototipo en zona piloto (10 sensores)
  2. Fase 2 (6 meses): Escalado a parque municipal completo
  3. Fase 3 (12 meses): Implementación en red de parques urbanos
  4. Fase 4 (18 meses): Plataforma abierta para otras ciudades
